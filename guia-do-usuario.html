🛡️ WAC Shield PRO — Guia do Usuário 1. Requisitos do Sistema WordPress: 6.0 ou superior. PHP: 7.4 ou superior. Desenvolvedor: WAC Security Solutions. Licença: Token de acesso ativo (fornecido pela WAC Security Solutions). 2. Instalação Inicial Fazer Upload: No painel do WordPress, vá em Plugins > Adicionar Novo > Enviar Plugin. Selecionar Arquivo: Escolha o arquivo wac-shield-pro.zip e clique em Instalar Agora. Ativar: Clique em Ativar Plugin. O sistema criará automaticamente as tabelas de segurança de alto desempenho no seu banco de dados. Acesso: Um novo menu chamado WAC Shield aparecerá na sua barra lateral do WordPress. 3. Como Atualizar (Obrigatório) O WAC Shield PRO possui um sistema de atualização inteligente via nuvem desenvolvido pela WAC Security Solutions. Você tem dois caminhos para manter seu plugin protegido: Caminho 1 (Painel Geral): Vá em Painel > Atualizações. Se houver uma nova versão, o WAC Shield aparecerá na lista. Basta marcar a caixa e clicar em Atualizar Plugins. Caminho 2 (Lista de Plugins): Vá em Plugins > Plugins Instalados. Se uma atualização estiver disponível, você verá um aviso logo abaixo do nome do WAC Shield PRO. Clique em "Atualize agora". Nota: Se você sabe que uma versão nova foi lançada e ela ainda não apareceu, vá em Painel > Atualizações e clique no botão "Verificar novamente" para forçar a leitura da nuvem. 4. Configuração de Defesa Token de Acesso: Insira seu Token PRO no campo de licenciamento para ativar a proteção em tempo real fornecida pela WAC Security Solutions. País Autorizado: Selecione seu país principal (ex: Brasil 🇧🇷). Apenas usuários com IP originado deste local conseguirão acessar a tela de login. Força Bruta: Defina o limite de tentativas (recomendado: 3). O tempo de banimento aumenta automaticamente a cada nova tentativa detectada. Lista de Permissões (Whitelist): > ⚠️ CRÍTICO: Sempre adicione seu endereço IP atual na Whitelist. Isso garante que você nunca seja bloqueado pelo sistema, funcionando como uma "Chave Mestra" de acesso. 5. Resolução de Problemas Mensagem "Acesso Negado": Verifique se o seu Token está correto ou se a sua licença expirou. Entre em contato com o suporte da WAC Security Solutions. IP Bloqueado por Engano: Caso algum usuário legítimo seja bloqueado, você pode remover o IP dele diretamente na tabela de "Alertas de Segurança Recentes" no painel do plugin. Desempenho: O plugin utiliza tabelas SQL dedicadas, garantindo o máximo de proteção sem comprometer a velocidade do seu site.
